# Broadcomm BCM2835 (Raspberry Pi Zero/Zero W) and ST7735, 80x160 display

## Wiring

| Pico Pin  | GPIO Pin | ST7735 Board |
|-----------|----------|--------------|
| 3 - GND   | n/a      | GND          | 
| 36 - VCC  | n/a      | VCC          | 
| 14        | 10       | SCL          | 
| 15        | 11       | SDA          | 
| 17        | 13       | CS           | 
| 19        | 14       | DC           | 
| 20        | 15       | RES          |
